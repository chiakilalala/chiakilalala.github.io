let const var 差別 - var 變數的 scope 以函式為界線，迴圈內外屬於同一個 scope scope 可用範圍 - let 可用範圍以程式區塊 ( 大括號 ) 為分界線 let 以變數scope for(let i=0; i<5;i++){ console.log(i); } - const 宣告常數 常數資料不能變動
